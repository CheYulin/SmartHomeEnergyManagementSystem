调用输入：
==========
Tout：外界温度（24/调度间隔）K个点；
Price：电价k个点；
States：包含3个数据，事件发生事件tk，tk时刻对应的室内温度和热水器温度（lastTac,lastTwh）（可以从上次的优化结果中查找，如果是第一次优化，向优化程序传送默认值）；
Rac，Rwh：空调需求和热水器需求，可包含多个（最多4条），格式

		需求开始时刻，需求结束时刻，需要温度，控制偏差 （换行）
		需求开始时刻，需求结束时刻，需要温度，控制偏差


运行结果：
==========
将保存从tk开始至K(终止时刻)的变量值，包括：

qac,qwh, STac, STwh
即空调功率，热水器功率，室内温度，热水温度（后两个用来展现结果和对lastTac和lastTwh赋值）
